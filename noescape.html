No Escape

  • Year: Sophomore
  • Project URL: Link to GitHub

No Escape is game that uses 3D mazes to simulate the point of view of a mouse inside a maze, made purely in Java. I created this project with the help of a junior during Code211, a one day hackathon for my school district, and won best project in the intermediate category. The maze data is stored through a two dimensional array that is later rendered during runtime. The goal is to navigate to the center where there is teleporter to bring you to the next level. The game was too hard to complete when testing it with students at the event so we also added a minimap to help. The minimap is an empty image created on compile time. The code iterates over the maze data and places black pixels, signifying walls, on the minimap in the coresponding locations. This makes it so the minimap works independently to what maze is given, opening the possibility to randomly generating mazes and an infinite amount of levels.
